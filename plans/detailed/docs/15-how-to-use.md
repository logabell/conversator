
# How to Use This Spec Pack

1. Start with `docs/00-overview.md` and `docs/03-architecture.md`.
2. Implement workspace layout from `docs/11-workspace-layout.md`.
3. Implement event-sourced state from `docs/04-task-session-state.md`.
4. Build prompt refinement loop from `docs/06-workflows.md` + templates.
5. Integrate Beads (`docs/12-phases-and-validation.md` Phase 3).
6. Add builder adapter (`docs/12-phases-and-validation.md` Phase 4).
7. Enforce memory budgets (`docs/05-memory-and-rag.md`).
8. Add UX polish and gates (`docs/10-ui-ux.md` + security doc).

When in doubt:
- store pointers, not prose
- freeze only final handoffs
- attach evidence to Beads
